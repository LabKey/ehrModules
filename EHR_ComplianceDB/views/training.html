• This page provides a mechanism for WNPRC Trainers to directly enter the dates that staff complete training requirements.

  • Below are three grids. The top grid displays a summary of each WNPRC employee, the requirements they need to complete, and the last date they completed it.

  • The second grid displays the dates that each employee completed Complaince/Training requirements. You can import new information using the 'Insert New' button. Employees must be entered using their employee ID. This is usually the beginning of their WNPRC email.

  • At the bottom of this page there is a list of all WNPRC employees, including their employee ID. This can be used to look up an employee ID if you do not know it.

  • All grids can be sorted and filtered by clicking on the column header.
